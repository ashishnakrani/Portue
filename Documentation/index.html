Get started

Item name: Aerik - One page portfolio template

The template folder Aerik is available inside the downloaded zip file. Need to unzip the zip file to find the template also the documentation folder.

  1. css — folder with CSS files.
  2. fonts — folder with fonts files.
  3. img — folder with image files.
  4. js — folder with Javascript files.
  5. php — folder with form-process.php files.

Upload the template files to the server with the help of one of the FTP-clients like FileZilla.
